# 强化版 swagger-markdown-ui



## 简介

将Swagger V2 文档转换为离线的 Markdown 格式的文档




## 特点

- ~~支持swagger V3~~（仅支持我自己使用的版本，大家想用自己适配一下吧）
- 通过URL获取接口
- 格式化文档结构
- 完整汉化
- 提取通用响应内容
- 更完整的响应内容




## 在线使用
[swagger-markdown-ui](http://swagger-markdown-ui.conststar.cn)




## 实例

## 快速使用

#### 初始化项目
```
npm install
```

#### 编译并运行
```
npm run serve
```

#### 编译生成
```
npm run build
```

#### 运行测试
```
npm run test
```

#### Lints and fixes files
```
npm run lint
```
